# 烂梗喵

一个 Windows 桌面工具：从 [sb6657](https://sb6657.cn) 获取随机烂梗，并通过可配置按键在 CS2 中发送。

## 当前功能

- Godot 4 C# 图形化界面
- 调用 `GET /machine/getRandOne` 获取随机烂梗
- 维护少量预取队列，降低按键发送延迟
- 可配置触发键，默认 `F8`
- 可配置全体聊天 / 队内聊天键，默认 `Y`
- 剪贴板粘贴 + Windows 输入模拟发送
- CS2 前台窗口检测
- 发送历史、接口测试、配置持久化

## 开发环境

- Godot .NET 4.6.x
- .NET 8 SDK
- Windows

使用 Godot .NET 编辑器打开项目目录即可运行。也可以使用对应版本的 `dotnet` / Godot 构建命令。

当前工作区已经包含本地开发工具时，可以直接双击 `run.cmd`。启动脚本会自动设置项目内的 .NET 8 路径和可写数据目录，避免直接启动 Godot 时因运行时或日志目录不可用而崩溃。

## 发送方式说明

本项目不注入 CS2、不读取游戏进程内存，也不使用 `-allow_third_party_software`。发送功能通过剪贴板和 Windows 键盘输入模拟完成。

自动化输入可能受到 Steam 或服务器规则限制，请先在离线、私人或你确认允许的环境中测试。触发键仅在 CS2 位于前台时执行。

## sb6657 接口约定

第三方客户端不携带 sb6657 官网专用的来源统计请求头。程序只使用公开随机烂梗接口的 JSON 响应。
